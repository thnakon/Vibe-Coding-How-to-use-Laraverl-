Mail & Notifications

สร้างความผูกพันกับผู้ใช้ด้วยการส่งอีเมลและการแจ้งเตือนที่เป็นระบบ

1. การสร้าง Mailable

Mailable คือ Class ที่ใช้กำหนดเนื้อหาและรูปแบบของอีเมล

php artisan make:mail WelcomeMail

ตัวอย่างการตั้งค่าใน app/Mail/WelcomeMail.php:

public function content(): Content
{
    return new Content(
        view: 'emails.welcome', // กำหนดหน้า Blade ที่จะใช้เป็นเนื้อหาเมล
        with: ['name' => $this->user->name],
    );
}

2. การเพิ่มปุ่ม (Buttons/Markdown)

ถ้าคุณใช้ Markdown Mail คุณจะได้ดีไซน์เมลที่สวยงามและรองรับ Responsive อัตโนมัติ

php artisan make:mail OrderShipped --markdown=emails.orders.shipped

3. การส่งเมล

use Illuminate\Support\Facades\Mail;
use App\Mail\WelcomeMail;

Mail::to($user->email)->send(new WelcomeMail($user));

4. ระบบ Notifications

เจ๋งกว่าเมล! เพราะส่งได้ทั้ง Mail, Database (แสดงหน้าเว็บ), และ Slack พร้อมกันในคำสั่งเดียว

php artisan make:notification InvoicePaid

// การเรียกใช้งาน
$user->notify(new InvoicePaid($invoice));
Tip: ใช้ Queue เข้ามาช่วยในการส่งเมล เพื่อไม่ให้หน้าเว็บค้างรอจนเมลส่งเสร็จ จะทำให้ User Experience ดีขึ้นมากครับ
