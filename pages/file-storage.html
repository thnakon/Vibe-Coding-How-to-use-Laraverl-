File Storage

จัดการการอัปโหลดไฟล์และรูปภาพอย่างปลอดภัยด้วยระบบ Filesystem ของ Laravel

1. Storage Link

ตามปกติไฟล์ที่อัปโหลดจะถูกเก็บไว้ในโฟลเดอร์ storage ซึ่งเข้าถึงจากหน้าเว็บไม่ได้ เราต้องรันคำสั่งเชื่อมโยงไปที่ public

php artisan storage:link

2. การอัปโหลดไฟล์

// ใน Controller
public function store(Request $request) {
    if ($request->hasFile('avatar')) {
        $path = $request->file('avatar')->store('avatars', 'public');
        // $path จะได้ค่าประมาณ 'avatars/random_name.jpg'
        return $path;
    }
}

3. การเรียกใช้ไฟล์ใน Blade

<img src="{{ asset('storage/' . $user->avatar) }}" alt="Avatar">

4. การลบไฟล์

use Illuminate\Support\Facades\Storage;

Storage::disk('public')->delete($user->avatar);
ข้อแนะนำ: คุณสามารถเปลี่ยน disk เป็น s3 (Amazon S3) ในไฟล์ .env ได้ทันทีโดยไม่ต้องแก้โค้ดเลย!
