Localization (i18n)

สร้างเว็บไซต์ที่รองรับได้หลายภาษาด้วยระบบ Translation ของ Laravel

1. การตั้งค่าภาษาหลัก

คุณสามารถกำหนดภาษาเริ่มต้นของระบบได้ในไฟล์ config/app.php ที่บรรทัด 'locale' => 'th'

2. การสร้างไฟล์คำแปล

สร้างโฟลเดอร์ภาษาใน lang/ เช่น lang/th/messages.php

// lang/th/messages.php
return [
    'welcome' => 'ยินดีต้อนรับสู่เว็บไซต์ของเรา',
    'login' => 'เข้าสู่ระบบ',
];

3. การเรียกใช้คำแปล

// ใน Blade
<h1>{{ __('messages.welcome') }}</h1>

// ใน Controller
echo __('messages.login');

4. การใส่พารามิเตอร์ในคำแปล

// ในไฟล์ภาษา
'greeting' => 'สวัสดีคุณ :name',

// ตอนเรียกใช้
{{ __('messages.greeting', ['name' => 'สมชาย']) }}
ข้อแนะนำ: คุณสามารถใช้ไฟล์ JSON แทน PHP array ได้ด้วย (เช่น lang/th.json) เพื่อความสะดวกในการจัดการคำแปลจำนวนมาก
