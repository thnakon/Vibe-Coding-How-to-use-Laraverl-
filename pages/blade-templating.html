Blade Templating & Components

Blade ไม่ใช่แค่ไฟล์ HTML ธรรมดา แต่มันคือระบบที่ช่วยให้เราสร้างเว็บไซต์ที่เป็นระเบียบและลดการเขียนโค้ดซ้ำซ้อน

1. Layout Inheritance (การสืบทอด Layout)

เราไม่ต้องเขียน Header หรือ Footer ซ้ำในทุกหน้า เพียงแค่สร้าง Master Layout ไว้

Master Layout (layouts/app.blade.php)

<html>
    <head>
        <title>@yield('title')</title>
    </head>
    <body>
        <nav>...</nav>
        
        <div class="container">
            @yield('content')
        </div>

        <footer>...</footer>
    </body>
</html>

Page Child (welcome.blade.php)

@extends('layouts.app')

@section('title', 'หน้าแรก')

@section('content')
    <h1>ยินดีต้อนรับสู่เว็บไซต์</h1>
    <p>เนื้อหาหน้าแรก...</p>
@endsection

2. Blade Components

Components คือการสร้างบล็อก HTML ที่ใช้งานซ้ำๆ ได้ เช่น ปุ่ม (Button) หรือ การ์ด (Card)

// ใช้ artisan สร้าง component
php artisan make:component Alert

การเรียกใช้งานใน HTML:

<x-alert type="error" message="เกิดข้อผิดพลาด!" />

3. Directives ที่ใช้บ่อย

Blade มีทางลัดให้เราเขียน PHP ใน HTML ได้สั้นลง

{{-- จัดการ Authentication --}}
@auth
    <p>สวัสดีคุณ {{ auth()->user()->name }}</p>
@else
    <a href="/login">เข้าสู่ระบบ</a>
@endauth

{{-- การวนลูปที่ปลอดภัย --}}
@forelse($users as $user)
    <li>{{ $user->name }}</li>
@empty
    <li>ไม่พบผู้ใช้งาน</li>
@endforelse
คำคม Blade: "Write less, do more" - Blade ถูกคอมไพล์เป็นโค้ด PHP ปกติ 100% จึงไม่มีผลเสียต่อประสิทธิภาพของเว็บ
